How to Use FormFaces

Using FormFaces is as simple as inserting a single line of code into any XForms document. Add the line

<script type="text/javascript" src="formfaces.js"></script>

anywhere in the <head/> section of your HTML page to enable full XForms support. (Change the path to formfaces.js if it is not in the same directory as your XForms page.)

Including the formfaces.js script file will automatically invoke the FormFaces processor, which will subsequently find and render the XForms elements and controls on the page. That's it—one line of code. It's that easy!
