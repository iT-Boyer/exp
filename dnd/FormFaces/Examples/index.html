FormFaces Examples

Bookstore

This page is a mock-up of a simple bookstore. It demonstrates many features on one page, including calculations, dynamically repeated tables, relevance, validation, and more. At the bottom of the bookstore is a link to a page explaining all of the features the bookstore demonstrates.

Mozilla Calculator

This is the XForms calculator the folks at Mozilla put together to demonstrate their processor. We made no modifications to this page to get it working in FormFaces aside from adding <script src="xforms.js"></script>. As you can see, FormFaces runs the Mozilla calculator flawlessly in Internet Explorer, Mozilla, and Opera!

Test Pages

We have test pages that demonstrate all of the XForms features that FormFaces supports. These test pages are primarily designed to test specific features, so don't expect fully-developed graphical demo pages.
