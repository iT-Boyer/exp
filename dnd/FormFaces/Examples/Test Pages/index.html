FormFaces Test Pages

Small test pages are provided to demonstrate how FormFaces processes and renders XForms functionality. These are ordered from most basic to most complex.

Keep in mind that these pages were created for testing purposes—meaning, they're not the most visually impressive pages, nor are they intended to look like a real tax form or online shopping cart.

output.html
The simplest page; demonstrates an output control bound to an instance data node.
io.html
A simple page demonstrating that two controls bound to the same instance data node are linked.
controls.html
Demonstrates controls.
group.html
Demonstrates the group element, particularly nested ref attributes.
switch.html
A simple page demonstrating a switch with three cases, and a button to toggle between them.
outputValue.html
Verifies that <output value="..."/> works.
namedBind.html
Demonstrates that named and anonymous binds work the same.
namedBind2.html
A more complicated page demonstrating named/anonymous binds nested within each other.
itemset.html
Demonstrates the <itemset/> element, determining a list of selection choices based on the instance data.
relevance.html
A page demonstrating the relevant model item property and how relevance is inherited from ancestor nodes.
MIPs.html
Demonstrates Model Item Property events.
repeat.html
A simple demonstration of the repeat control.
nestedRepeats.html
An demonstration of nested repeats.
totals.html
A "customer database" page demonstrating nested repeats. Includes calculations and tests the XPath context of nested XPaths.
insert-delete.html
Demonstrates the insert and delete actions of repeats, as well as the index() XPath function.
vehicles.html
Demonstrates itemset, which allows a select control's choices to be drawn from the instance data.
trigger.html
Very simple demonstration of a trigger; opens Yahoo! in a new window.
increment.html
Demonstrates the DOMActivate event and the setvalue action with a button you can click to increment a number on the page.
dispatch.html
Demonstrates event dispatching.
xforms-ready.html
Demonstrates that the xforms-ready event is fired when the processor has loaded the page.
events.html
Demonstrates the firing of events, in particular DOMFocusIn and DOMFocusOut.
event-handler.html
Demonstrates usage of the ev:handler attribute.
submit.html
Demonstrates the submission process by loading data from an external XML file.
replace-all.html
Demonstrates the replace="all" option for submission, replacing the current page with the page returned from the server.
xformsFunctions.html
Demonstrating XForms Date/Time/Duration calculation functions.
submit-validation.html
Demonstrates checking if the data is valid before submission, an error will be thrown if it is invalid.
controls-labels.html
Demonstrating all controls working with labels that have refs.
